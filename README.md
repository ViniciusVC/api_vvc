# api_vvc

```
Experiments with API. 
Alguns testes de API. 
NODEJS + EXPRESS + nodemon
```

# Rodar o projeto :
```
npm star
```

# Comando que foram usados.

## Gerar o package.json :
```
npm init -y
```

## Instalar o NODEJS no diretorio do projeto:
```
npm install
```

## Instalar o EXPRESS
```
npm install express
(Framework web que ajuda a criar APIs)
```

## Instakar o NODEMON
